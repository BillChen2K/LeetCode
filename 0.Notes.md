# LeetCode Notes

## 递归、回溯、DFS

### Notes

```cpp
DFS（顶点） 
{
　　处理当前顶点，记录为已访问
　　遍历与当前顶点相邻的所有未访问顶点
　　{
　　　　　　标记更改;
　　　　　　DFS( 下一子状态);
　　　　　　恢复更改;
　　}
}
```

```cpp
queue<type> q;
q.push(初始状态);
while (!q.empty())
{
  type t = q.front() ;
  q.pop();
  遍历 t 的各个Next状态  next
  { 
    if (next is legal)
      q.push(next); 计数或维护等; 
  } 
}
```

#### [对称二叉树](https://leetcode-cn.com/problems/symmetric-tree/description/)

使用 BFS 检查每一层是否是对称的，也可以使用递归，连个指针分别向左、右走。

```cpp
bool solve(TreeNode *pa, TreeNode *pb) {
    if (!pa && !pb) {
        // Two null
        return true;
    }
    if (!pa || !pb) {
        // One of them is null
        return false;
    }
    return pa->val == pb->val && solve(pa->left, pb->right) && solve(pa->right, pb->left);
}
```
